## Prerequisites

- A CostHawk account (join the [waitlist](https://costhawk.ai) if you don't have one)
- Claude Code or Claude Desktop installed

## Step 1: Get Your Access Token

1. Log into your [CostHawk dashboard](https://costhawk.ai/dashboard)
2. Navigate to **Settings â†’ Developer**
3. Click **Create Token**
4. Copy the token (you'll only see it once)

<Warning>
  Keep your access token secret. Never commit it to version control or share it publicly.
</Warning>

## Step 2: Add to Claude Code

<Tabs>
  <Tab title="macOS/Linux">
    The fastest way to get started is with the one-liner command:

    <CodeGroup>

    ```bash Global (All Projects)
    claude mcp add -s user -e COSTHAWK_API_KEY=YOUR_TOKEN_HERE costhawk -- npx --yes costhawk
    ```

    ```bash Current Project Only
    claude mcp add -e COSTHAWK_API_KEY=YOUR_TOKEN_HERE costhawk -- npx --yes costhawk
    ```

    </CodeGroup>

    Replace `YOUR_TOKEN_HERE` with your access token from Step 1.
  </Tab>
  <Tab title="Windows">
    Run the setup wizard which handles everything automatically:

    ```powershell
    npx --yes costhawk@latest --setup
    ```

    This will prompt for your API key and configure everything for you.

    <Warning>
      The `claude mcp add` command has argument parsing issues on Windows. Use the setup wizard instead.
    </Warning>
  </Tab>
</Tabs>

## Step 3: Restart Claude Code

After adding the MCP server, restart Claude Code to load the new tools.

You can now ask Claude about your API costs:

## Alternative: Manual Configuration

If you prefer to configure manually, add this to your MCP config file:

<Tabs>
  <Tab title="Claude Code">
    Edit `~/.claude/.mcp.json`:
    ```json
    {
      "mcpServers": {
        "costhawk": {
          "command": "npx",
          "args": ["-y", "costhawk"],
          "env": {
            "COSTHAWK_API_KEY": "YOUR_TOKEN_HERE"
          }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Claude Desktop (macOS)">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json`:
    ```json
    {
      "mcpServers": {
        "costhawk": {
          "command": "npx",
          "args": ["-y", "costhawk"],
          "env": {
            "COSTHAWK_API_KEY": "YOUR_TOKEN_HERE"
          }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Claude Desktop (Windows)">
    Edit `%APPDATA%\Claude\claude_desktop_config.json`:
    ```json
    {
      "mcpServers": {
        "costhawk": {
          "command": "npx",
          "args": ["-y", "costhawk"],
          "env": {
            "COSTHAWK_API_KEY": "YOUR_TOKEN_HERE"
          }
        }
      }
    }
    ```
  </Tab>
</Tabs>
